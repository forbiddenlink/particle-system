Contact

Use this page when you need help with the demo, want to report rendering behavior, or want to discuss integration ideas. Include your browser version, operating system, and a short summary of the issue so triage stays fast and actionable.

For reproducible bugs, attach expected behavior, actual behavior, and exact control values used when the issue appeared. If the problem is visual, include a short screen recording and note whether WebGPU or fallback rendering was active.

Primary contact channels

  • Email: support@nova-particles.dev
  • Technical requests: devrel@nova-particles.dev
  • Security reports: security@nova-particles.dev

We aim to acknowledge incoming requests within two business days. Critical security concerns are prioritized. Please avoid sharing sensitive personal data in initial messages.
